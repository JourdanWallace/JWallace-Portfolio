Technologies Page

This is my Technologies page and I will describe how to use HTML and CSS tags.

The tags HTML and CSS are vitally important when inputting code. Hypertext Markup Language,or better known as HTML, is a standardized system for tagging text files to achieve font, color, graphic, and hyperlink effects on web pages.

Cascading Style Sheets, or CSS, is a style sheet language used for describing the look and formatting of a document written in a markup language. These are the tags used to change all of the details on a website, such as the background color, borders, margins, etc.
